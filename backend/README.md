# Codigo Certo API
## Tarefa (funcional)
Sistema para uma landing page que tem como funcionalidade: SAC, ROADMAP, VOLUNTARIADO
## Stack utilizada

TypeScript

Nest.js

TypeORM

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
